### ✨ Proyectos

**Hospital V&M · Fullstack I**
API REST de gestión hospitalaria con Spring Boot + MySQL. CRUD completo sobre pacientes, médicos, especialidades y atenciones. Incluye cálculo de sueldos con comisiones, cobertura FONASA/ISAPRE, validación de unicidad, filtros por fecha/edad/previsión y zona horaria Chile. Docker Compose listo, 25 tests unitarios, documentado con Swagger.

`Java · Spring Boot · MySQL · JPA/Hibernate · REST API · Maven · Docker · Swagger · JUnit · Mockito`

**Huerto Hogar · Fullstack II**
E-commerce de productos orgánicos con React y Firebase. Catálogo con imágenes de productos, carrito de compras, perfiles de cliente y administrador, registro de usuarios y reseñas.

`React · JavaScript · Firebase · CSS · HTML`

**OpenCodeTools**
Framework de persistencia local con integridad criptográfica para entornos multi-agente. Verificación cruzada entre Python y PowerShell, suite de +285 tests automatizados, búsqueda full-text y módulos autónomos de skill. Arquitectura cero dependencias cloud.

`Python · Node.js · PowerShell · SQLite · SHA-256 · FTS5`

**Colegio O'Higgins Platform · Fullstack III**
Plataforma de gestión académica con 6 microservicios (Spring Boot + React + MySQL). Descubrimiento con Eureka, API Gateway con JWT, Circuit Breaker en asistencia, 10 contenedores Docker, monitoreo con Prometheus/Grafana, pipeline CI/CD con 475 tests (unitarios + integración), cobertura con JaCoCo y saneamiento de deuda técnica documentado.

`Java · Spring Boot · React · MySQL · Docker · JWT · Eureka · Prometheus · Grafana · GitHub Actions · JaCoCo`
